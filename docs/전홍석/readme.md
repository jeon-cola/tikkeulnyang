## 게이미피케이션 공부
    국민은행에서 만든 "게이미피케이션을 활용한 국내외 금융 상품-서비스 적용사례" 를 통해서 게이미피케이션의 원리와 적용방법, 사례 등을 공부 했다.
    공부한 내용을 기반으로 프로젝트 내에서 적용하려고 노력함함
